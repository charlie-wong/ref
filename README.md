# ref
Show example usage of a given command, for quick reference<br>
The program's action is affected by the config file under `~/.config/ref/config` if it exist.<br>

# install & uninstall
`make install`<br>
`make uninstall`<br>

# Tips
Makefiles is under `build` directory.<br>
